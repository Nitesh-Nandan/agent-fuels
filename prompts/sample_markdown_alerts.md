# Sample Markdown — GitHub Alerts Demo

A quick reference for using GitHub-style alert blocks in your markdown files.

---

## 📌 What Are Alerts?

Alerts are special blockquotes that render with **color-coded icons** on GitHub and modern markdown viewers.
They help highlight important information at a glance.

---

## Alert Types

### 🔵 NOTE — Background Context

> [!NOTE]
> This is a note. Use it to share background context, supplementary information, or helpful explanations that aren't critical but add value.

---

### 🟢 TIP — Best Practices

> [!TIP]
> This is a tip. Use it to share performance optimizations, best practices, shortcuts, or efficiency suggestions.

---

### 🟣 IMPORTANT — Must-Know Info

> [!IMPORTANT]
> This is important. Use it for essential requirements, critical steps, or information the reader must not miss.

---

### 🟡 WARNING — Potential Problems

> [!WARNING]
> This is a warning. Use it for breaking changes, compatibility issues, or anything that could cause unexpected behavior.

---

### 🔴 CAUTION — High-Risk Actions

> [!CAUTION]
> This is a caution. Use it for high-risk actions that could cause data loss, security vulnerabilities, or irreversible changes.

---

## Real-World Example — Prompt File

Below is how you might use alerts inside a chapter prompt:

---

### Chapter: Morning Reflection

> [!IMPORTANT]
> Review this chapter every morning before starting your day. Consistency is key.

Start by taking 3 deep breaths and grounding yourself in the present moment.

> [!TIP]
> If you're feeling rushed, even 2 minutes of reflection is better than none.

Write down one intention for the day. Keep it simple and actionable.

> [!WARNING]
> Avoid checking your phone before completing this exercise. It disrupts focus.

End with a short gratitude note — just 1–2 sentences is enough.

> [!NOTE]
> This practice is adapted from the *Little Care* framework. Revisit the full guide monthly.

---

## Syntax Reference

```markdown
> [!NOTE]
> Your note text here.

> [!TIP]
> Your tip text here.

> [!IMPORTANT]
> Your important text here.

> [!WARNING]
> Your warning text here.

> [!CAUTION]
> Your caution text here.
```

> [!NOTE]
> These render natively on **GitHub**, **VS Code preview**, and **Antigravity artifacts**. In plain viewers, they appear as standard blockquotes.
